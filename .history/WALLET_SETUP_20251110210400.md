# üîê Wallet Setup Guide for VIVO RUN

## Required Wallet: Freighter

VIVO RUN uses **Freighter** wallet for Stellar blockchain integration. Freighter is the official browser extension wallet for the Stellar network.

## üì• Installation Steps

### Option 1: Chrome/Edge/Brave
1. Visit [https://freighter.app/](https://freighter.app/)
2. Click **"Download"** or **"Get Freighter"**
3. Click **"Add to Chrome"** (or your browser)
4. Click **"Add Extension"** in the browser prompt
5. The Freighter icon will appear in your browser toolbar

### Option 2: Firefox
1. Visit [https://freighter.app/](https://freighter.app/)
2. Click **"Download"** or **"Get Freighter"**
3. Click **"Add to Firefox"**
4. Click **"Add"** in the browser prompt
5. The Freighter icon will appear in your browser toolbar

## üöÄ First Time Setup

### 1. Create a New Wallet (or Import Existing)
- Click the Freighter extension icon in your browser
- Choose **"Create New Wallet"** or **"Import Existing Wallet"**
- If creating new:
  - Write down your **12-word recovery phrase** (keep it safe!)
  - Set a password
- If importing:
  - Enter your recovery phrase
  - Set a password

### 2. Switch to Testnet (for testing)
- Open Freighter extension
- Click the network dropdown (usually shows "Mainnet")
- Select **"Testnet"**
- This allows you to test without using real XLM

### 3. Get Testnet XLM (for testing)
- Visit [Stellar Laboratory - Account Creator](https://laboratory.stellar.org/#account-creator?network=test)
- Enter your Stellar address (from Freighter)
- Click **"Fund Account"**
- You'll receive 10,000 testnet XLM (free, for testing only)

## ‚úÖ Verify Installation

1. **Check Extension is Installed**
   - Look for the Freighter icon in your browser toolbar
   - Click it to open the wallet

2. **Test Connection in VIVO RUN**
   - Open VIVO RUN in your browser
   - Click **"Connect Stellar Wallet"** button
   - You should see a Freighter popup asking for permission
   - Click **"Allow"** or **"Connect"**
   - Your wallet address should appear in the button

## üéÆ Using with VIVO RUN

### Connecting Your Wallet
1. Click **"Connect Stellar Wallet"** in the game menu
2. Freighter will show a connection request
3. Click **"Allow"** to connect
4. Your wallet address will be displayed (e.g., `GABC...XYZ1`)

### Disconnecting
- Click the wallet button again (it will show your address)
- Click **"Disconnect"** or click the button to disconnect

## üîß Troubleshooting

### Button Not Working?
1. **Check if Freighter is installed**
   - Look for the Freighter icon in your browser toolbar
   - If not visible, reinstall from [freighter.app](https://freighter.app/)

2. **Check Browser Permissions**
   - Right-click Freighter icon ‚Üí "Manage Extension"
   - Ensure it has permission to access the website
   - Try refreshing the page

3. **Check Console for Errors**
   - Press `F12` to open Developer Tools
   - Go to "Console" tab
   - Look for error messages starting with üîå or ‚ùå

4. **Try Different Browser**
   - Freighter works best with Chrome, Edge, or Firefox
   - Try a different browser if issues persist

### Connection Rejected?
- Make sure you click "Allow" in the Freighter popup
- Check that Freighter is unlocked (enter password if needed)
- Try disconnecting and reconnecting

### No Testnet XLM?
- Visit [Stellar Laboratory](https://laboratory.stellar.org/#account-creator?network=test)
- Enter your address and click "Fund Account"
- Wait a few seconds for the XLM to arrive
